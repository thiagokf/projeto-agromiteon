# Website da Agromiteon
## Frontend
Nesse diretorio reside o frontend do website que desenvolvi para atender os interesses da Agromiteon.
### tecnologias
Para desenvolver o frontend, utilizei do HTML, CSS e Javascript. Decidi usar essas tecnologias pois tinha mais segurança na epoca, e tambem por ser apenas uma lading page simples, que não exigia muitas paginas e funções. O Javascript foi utilizado tanto para dar movimento pro site quanto para pegar e mandar as informações do form pro backend, que esta no diretorio *agromiteon-serv*
