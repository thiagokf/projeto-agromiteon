# Site da Agromiteon - Frontend
Nesse diretório, reside o frontend do website que desenvolvi para atender os interesses da Agromiteon.
## Tecnologias Utilizadas
- Html e Css
- Javascript

Decidi usar essas tecnologias pois tinha mais segurança na época, e também por ser apenas uma landing page simples, que não exigia muitas paginas e funções. O Javascript foi utilizado tanto para dar movimento pro site quanto para pegar e mandar as informações do form pro backend, que está no diretório [***agromiteon-serv***](https://github.com/thiagokf/agromiteon-server)
